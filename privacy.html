Privacy Policy

Isolyr Labs

Last updated: December 19, 2025

Introduction

As a development laboratory, Isolyr Labs collects only the minimum data required for the operation and maintenance of its systems.

This policy describes what data is collected, how it is used, and the third-party services involved.

1. Data Collection

The specific data collected depends on which services you use:

Discord Bots:

  • User IDs and usernames (as provided by Discord)
  • Server IDs and server metadata
  • Message content when required for bot functionality
  • Command usage and interaction data

Minecraft Plugins:

  • Player UUIDs and usernames (as provided by Mojang)
  • In-game actions and statistics relevant to plugin functionality
  • Server configuration data

Custom Contracted Services:

  • Data collection is defined on a per-contract basis
  • Only data essential to service delivery is collected

Technical Metadata:

  • Timestamps of interactions
  • Error logs and diagnostic information
  • Service usage patterns for performance optimization

2. Data Usage

Data collected by Isolyr Labs is used exclusively for the following purposes:

  • Providing the core functionality of our services
  • Maintaining and improving service performance
  • Debugging and resolving technical issues
  • Preventing abuse, fraud, or malicious activity
  • Complying with legal obligations when required

We explicitly do not:

  • Sell user data to third parties
  • Use data for advertising or marketing purposes
  • Share data with third parties except as described in this policy

3. Third-Party Services

Isolyr Labs services operate on or interact with third-party platforms. Data handling on these platforms is governed by their respective privacy policies:

Discord:

  • Our Discord bots access user and server data through Discord's API
  • Discord's Privacy Policy applies to data stored on Discord's servers
  • We have no control over Discord's data handling practices

Minecraft / Mojang:

  • Player data (UUIDs, usernames) is provided by Mojang's authentication system
  • Mojang's Privacy Policy governs this data

GitHub and Development Platforms:

  • Used for code hosting, version control, and collaboration
  • Subject to their respective privacy policies

We encourage users to review the privacy policies of these third-party platforms.

4. Data Storage

Data storage practices vary by service:

  • Most data is stored temporarily in memory and not persisted to disk
  • Persistent data (such as user preferences or configurations) is stored securely
  • Logs and diagnostic data are retained only as long as necessary for troubleshooting
  • Data is not retained longer than required for its intended purpose

We implement reasonable security measures to protect stored data, but no storage system is completely secure.

5. User Rights

Depending on your jurisdiction and applicable laws, you may have the following rights:

  • Access: Request information about what data we hold about you
  • Correction: Request correction of inaccurate or incomplete data
  • Deletion: Request deletion of your data, subject to legal and operational constraints
  • Objection: Object to certain types of data processing

To exercise these rights, contact Isolyr Labs via Discord. We will respond to legitimate requests within a reasonable timeframe.

Note that some data may be retained if required by law or necessary for service operation.

6. Policy Changes

Isolyr Labs reserves the right to modify this Privacy Policy at any time. Changes will be reflected on this page with an updated revision date.

Significant changes may be communicated through service announcements where feasible. Continued use of our services after changes constitutes acceptance of the updated policy.

We recommend reviewing this policy periodically to stay informed about how we protect your data.

7. Contact

All communication with Isolyr Labs regarding this Privacy Policy is handled exclusively through Discord.
